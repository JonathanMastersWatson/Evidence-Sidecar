# Selective Disclosure

This folder defines how evidence may be disclosed **without over-exposure**.

Selective disclosure is a core safety property.

---

## Role in the Architecture

Selective disclosure ensures:

- proportional transparency,
- minimal revelation,
- defensible refusal,
- and resistance to coercive overreach.

Disclosure does not imply compliance or correctness.
